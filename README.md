⌨️ Terminal Portfolio - Chester OS
Un portfolio personal interactivo con estética de terminal retro, construido bajo los principios de Clean Code y Arquitectura Modular utilizando TypeScript y Vite.

🚀 Características
Simulación de Terminal: Efecto de escritura asíncrona (Typewriter effect) y auto-scroll automático.

Arquitectura Desacoplada: Separación clara entre la lógica de datos (commands.ts), las utilidades de control de tiempo (utils/) y la capa de renderizado de UI.

TypeScript Nativo: Uso de interfaces y tipado estricto para asegurar la escalabilidad del proyecto.

Diseño Minimalista: Estética de consola de comandos con soporte para links interactivos y visualización de proyectos.

Despliegue Continuo: Configurado para actualización automática mediante Netlify.

🛠️ Stack Tecnológico
Vite: Herramienta de construcción ultra rápida.

TypeScript: Lenguaje principal para una lógica robusta.

CSS3: Variables dinámicas y animaciones de keyframes para el efecto de cursor.

HTML5: Estructura semántica para accesibilidad.
